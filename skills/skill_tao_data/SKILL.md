# Skill: skill-tao-data

## 描述

TradingAgents-OpenClaw 数据层 Skill，提供统一的股票数据获取接口。支持多供应商自动故障转移，内置缓存机制。

## 功能

- 股票价格数据 (OHLCV)
- 财务报表数据 (资产负债表、利润表、现金流量表)
- 实时行情数据
